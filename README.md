# Webpack boilerplate

A minimalistic webpack boilerplate project.

## Prerequisites

[![node][node]][node-url]
[![npm][npm]][npm-url]
      
- [Node.js](https://nodejs.org)

## Commands

- `npm start` - start the dev server
- `npm run build` - create build in `dist` folder
